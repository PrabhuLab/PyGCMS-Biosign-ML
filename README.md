# PyGCMS-Biosign-ML
Biosignature detection and Chemical Characterization of PyGCMS Samples

Code and Processed Data for Wong, Prabhu et al. 2025
Organic geochemical evidence for life in Archean rocks identified by pyrolysis–GC–MS and supervised machine learning

Code folder contains preprocessed dataset for 406 samples and code files for all 4 major models.
